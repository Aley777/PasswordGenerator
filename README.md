# Password Generator

React ile geliştirilmiş, kullanıcı tercihlerine göre rastgele şifre oluşturan basit ve modern bir web uygulaması.

## Özellikler

- Şifre uzunluğu seçme
- Büyük harf, küçük harf, sayı ve sembol seçenekleri
- Tek tıkla şifre kopyalama
- Modern responsive arayüz
- React Hooks kullanımı

## Kullanılan Teknolojiler

- React
- Vite
- CSS

## Kurulum

```bash
npm install
npm run dev
